## ✨ What is Flow?

Flow is a stunning, modern GitHub Pages showcase and web interface theme for transfer.sh-web. It provides a beautiful, professional landing page that demonstrates the power and elegance of the transfer.sh file sharing service.

## 🎯 Design Philosophy

Flow embodies the concept of seamless, effortless file sharing - like water flowing smoothly. The design emphasizes:

- **Simplicity** - Clean, uncluttered interface
- **Fluidity** - Smooth animations and transitions
- **Elegance** - Beautiful gradients and modern aesthetics
- **Accessibility** - Works for everyone, everywhere

## 🌈 Visual Identity

- Primary: `#60a5fa` (Light Blue)

## 🚀 Quick Start

### View Locally

```bash
cd docs
python3 -m http.server 8000
# Visit http://localhost:8000
```

### Deploy to GitHub Pages

1. Enable GitHub Pages in repository settings
2. Set source to "GitHub Actions"
3. Push to main branch
4. Visit: <https://lusky3.github.io/transfer.sh-web/>

## 🎨 Customization

### Change Colors

Edit CSS variables in `docs/styles.css`:

```css
:root {
  --color-primary: #3b82f6;
  --gradient-primary: linear-gradient(135deg, #3b82f6 0%, #8b5cf6 100%);
}
```

### Update Content

Edit `docs/index.html`:

- Hero section text
- Feature descriptions
- Tech stack items
- Download links

### Modify Branding

The theme name "Flow" appears in:

- Page title and meta tags
- Navigation logo
- Footer branding
- Copyright notice
